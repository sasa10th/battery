# battery

## 프로젝트 주제 : 환경 요인별 조류 분포에 미치는 영향 분석
* 모둠원 이름: 이홍균, 신재윤, 김지환
* 선정한 도서: 도시를 바꾸는 새
* 탐구 동기: 새는 육상 생태계의 일부로서, 생태계의 보존에 있어 필수적인 역할을 수행한다. 최근 도시환경의 확장이 가속됨에 따라 조류의 서식 분포 또한 큰 영향을 받고 있다. SDG 15에서 말하는 육상 생태계의 보존을 따르는 한, 도시환경 조성과 도시 개발 과정에서는 이를 고려하는 것이 필수적인 요소로 다가온다. 이를 위해서는 조류의 분포에 영향을 미치는 환경 요인을 분석하는 것이 필수적이다. 
* 탐구 문제: 본 프로젝트는 인공지능 모델에 기반, 환경 요인에 따른 조류 분포 예측 모델을 제작하여 도시 환경 게획에 활용 및 파라미터 분석을 통한 환경 요인 중요도 분석이 가능토록 한다.
* 관련 지속 가능 발전 목표: SDG 15, 육상 생태계 보존
* 입력 값 / 출력 값: 환경 요인 () -> 조류 분포 정도
